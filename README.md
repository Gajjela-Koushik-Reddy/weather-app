# Weather App
1. clone repo: `git@github.com:Gajjela-Koushik-Reddy/weather-app.git`
2. `cd weather-app`
3. create env file: `touch .env`
4. add:  
`VITE_WEATHER_API_KEY = ***********************`   
`VITE_GEO_LOCATION_API_KEY = ******************`
5. run: `npm run dev`
